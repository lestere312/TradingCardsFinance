This is a mock up of our website to utilize our backend database.
Each page represents a table in our database besides this landing page.
This example currently only has one deck and collection but future
examples will have multiple of each.
The Cards page is a list of all the cards in our database.
A user can add cards at the bottom of their collection and deck page.
Next to the card on both deck and collection there is an option to delete
the card from that table.
The account page will show information about what the overall account owns.
